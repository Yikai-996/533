Breakout Trading Strategy: Bollinger Bands

Author

Yikai Li

1. Strategy Logic

The core concept of this breakout strategy relies on volatility expansion. We utilize Bollinger Bands to establish a dynamic threshold for price movement. When the asset’s closing price strongly pierces the upper Bollinger Band, it signals a potential bullish breakout driven by sudden momentum, prompting a long entry.

2. Asset Selection

For this analysis, I selected TSLA (Tesla Inc.). We screen for highly liquid assets that exhibit distinct volatility cycles. TSLA has demonstrated frequent and clear breakout patterns over the past two years, making it an ideal candidate to test momentum-based expansion strategies.

3. Breakout Definition & Risk Management

A long breakout is strictly defined as the daily closing price closing greater than the Upper Bollinger Band (calculated using a 20-day Simple Moving Average and a 2-standard-deviation width). - Timeout Period: If the trade does not exit profitably within 10 trading days, it will be automatically closed at the market price. - Stop-Loss Logic: A hard stop-loss is set at 2% below the entry price to strictly limit downside risk. Note on Parameter Selection: Instead of using a walk-forward optimization which might lead to curve-fitting on this specific asset, we opted to use the industry-standard Bollinger Band parameters (20-day SMA, 2-Std Dev). Testing these static parameters across the entire 2-year window helps demonstrate the strategy’s raw robustness without over-optimizing for historical noise.

5. Breakout Detection & Trade Simulation

In this section, we define the core breakout logic. We calculate the Bollinger Bands and simulate trades based on our predefined rules: a 10-day timeout and a 2% stop-loss.

Code 
import numpy as np

# ==========================================
# STRATEGY PARAMETERS (Explicitly Defined)
# ==========================================
BB_WINDOW = 20          # 20-day Simple Moving Average
BB_STD = 2.0            # 2 Standard Deviations for Upper Band
TIMEOUT_DAYS = 10       # Maximum days to hold a position
STOP_LOSS_PCT = 0.02    # 2% Stop-loss threshold
POSITION_SIZE = 100     # Shares per trade
# ==========================================

def detect_breakouts_and_trade(df, window, std_dev, timeout, stop_loss, size):
    # 1. Calculate Bollinger Bands
    df = df.copy()
    df['SMA'] = df['close'].rolling(window=window).mean()
    df['STD'] = df['close'].rolling(window=window).std()
    df['Upper_BB'] = df['SMA'] + (std_dev * df['STD'])
    
    trades = []
    in_position = False
    entry_price = 0.0
    entry_date = None
    days_in_trade = 0
    
    # 2. Iterate through data to simulate trades
    for i in range(len(df)):
        row = df.iloc[i]
        current_date = df.index[i]
        
        # Skip initial rows where Upper_BB is NaN
        if pd.isna(row['Upper_BB']):
            continue
            
        if not in_position:
            # Entry Signal: Close price strictly greater than Upper BB
            if row['close'] > row['Upper_BB']:
                in_position = True
                entry_price = row['close']
                entry_date = current_date
                days_in_trade = 0
        else:
            days_in_trade += 1
            
            # Exit Condition 1: Stop-Loss Triggered
            if row['close'] <= entry_price * (1 - stop_loss):
                trades.append({
                    'Entry Date': entry_date,
                    'Exit Date': current_date,
                    'Entry Price': round(entry_price, 2),
                    'Exit Price': round(row['close'], 2),
                    'Position Size': size,
                    'Direction': 'Long',
                    'Outcome': 'Stop Loss Triggered'
                })
                in_position = False
                
            # Exit Condition 2: Time Timeout
            elif days_in_trade >= timeout:
                profit_status = 'Timeout (Profit)' if row['close'] > entry_price else 'Timeout (Loss)'
                trades.append({
                    'Entry Date': entry_date,
                    'Exit Date': current_date,
                    'Entry Price': round(entry_price, 2),
                    'Exit Price': round(row['close'], 2),
                    'Position Size': size,
                    'Direction': 'Long',
                    'Outcome': profit_status
                })
                in_position = False

    return pd.DataFrame(trades), df

# Execute the strategy
trade_ledger, processed_data = detect_breakouts_and_trade(
    data, BB_WINDOW, BB_STD, TIMEOUT_DAYS, STOP_LOSS_PCT, POSITION_SIZE

6. Performance Metrics & Visualizations

In this final section, we analyze the fate of every backtested trade and evaluate the strategy’s overall performance using the required metrics.

Code 
import plotly.express as px
import pandas as pd
import numpy as np

# Ensure the ledger is not empty
if len(trade_ledger) > 0:
    
    # ==========================================
    # A. Trade Outcome Analysis (Histogram)
    # ==========================================
    outcome_counts = trade_ledger['Outcome'].value_counts().reset_index()
    outcome_counts.columns = ['Outcome', 'Count']
    
    fig_hist = px.bar(outcome_counts, x='Outcome', y='Count', 
                  title='Trade Outcome Distribution',
                  color='Outcome',
                  color_discrete_map={
                      'Timeout (Profit)': '#2ecc71',   # Green
                      'Stop Loss Triggered': '#e74c3c', # Red
                      'Timeout (Loss)': '#f39c12'       # Orange
                  })
    fig_hist.update_layout(xaxis_title="Fate of Trade", yaxis_title="Number of Trades")
    fig_hist.show()

    # ==========================================
    # B. Performance Metrics Calculation
    # ==========================================
    # Calculate percentage return per trade
    trade_ledger['Return_Pct'] = (trade_ledger['Exit Price'] - trade_ledger['Entry Price']) / trade_ledger['Entry Price']
    
    avg_return = trade_ledger['Return_Pct'].mean()
    
    # Sharpe Ratio Calculation
    # Assumption: 4% annual risk-free rate, approx 252 trading days/year
    RF_ANNUAL = 0.04
    years_in_backtest = 2
    trades_per_year = len(trade_ledger) / years_in_backtest
    
    # Calculate Excess Returns (Trade-based)
    excess_returns = trade_ledger['Return_Pct'] - (RF_ANNUAL / trades_per_year)
    
    if excess_returns.std() != 0:
        # Annualized Sharpe based on trade frequency
        sharpe_ratio = np.sqrt(trades_per_year) * (excess_returns.mean() / excess_returns.std())
    else:
        sharpe_ratio = 0.0

    print("==========================================")
    print("📈 STRATEGY PERFORMANCE SUMMARY")
    print("==========================================")
    print(f"Total Trades Executed: {len(trade_ledger)}")
    print(f"Average Return per Trade: {avg_return * 100:.2f}%")
    print(f"Annualized Sharpe Ratio: {sharpe_ratio:.2f} (Assumes {RF_ANNUAL*100:.0f}% Risk-Free Rate)")
    print("==========================================")
    
    # ==========================================
    # C. Strategy Equity Curve
    # ==========================================
    # Calculate cumulative profit in USD based on POSITION_SIZE
    trade_ledger['Profit_USD'] = (trade_ledger['Exit Price'] - trade_ledger['Entry Price']) * trade_ledger['Position Size']
    trade_ledger['Cumulative_Profit'] = trade_ledger['Profit_USD'].cumsum()
    
    fig_curve = px.line(trade_ledger, x='Exit Date', y='Cumulative_Profit', 
                   title='Strategy Equity Curve (Cumulative USD Profit)',
                   markers=True)
    fig_curve.update_layout(xaxis_title="Trade Exit Date", yaxis_title="Cumulative Profit ($)")
    fig_curve.show()

else:
    print("No trades were executed. Please adjust parameters to generate signals.")
